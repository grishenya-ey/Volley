# VolleyScore Master

**VolleyScore Master** — это десктопное приложение для управления счетом волейбольных матчей с интеграцией с vMix для трансляций.

---

## 📖 Общее описание

Приложение предназначено для ведения счета волейбольных матчей и автоматического обновления графики в программе vMix. Оно позволяет:

- ✅ **Ведение счета матча в реальном времени** — начисление очков, управление подачей, завершение партий
- ✅ **Автоматическое обновление данных в vMix** через HTTP API — динамические GT-инпуты, управление оверлеями
- ✅ **Управление составами команд и логотипами** — загрузка, редактирование, экспорт/импорт
- ✅ **Удаленное управление с мобильных устройств** — встроенный HTTP-сервер, QR-код для быстрого подключения
- ✅ **Ведение расширенной статистики** — атака, блок, подачи, ошибки соперника
- ✅ **Сохранение матчей в файлы** — JSON формат, автосохранение
- ✅ **ТВ-табло для второго экрана** — крупное отображение счета, синхронизация в реальном времени
- ✅ **Тайм-ауты** — 2 тайм-аута на партию для каждой команды, таймер 30 секунд

### Варианты волейбола

Приложение поддерживает несколько вариантов волейбола с разными правилами:

| Вариант | Побед в партиях | Максимум партий | Очки (обычные/решающая) |
|---------|-----------------|----------------|------------------------|
| **Зал** | 3 | 5 | 25 / 15 |
| **Пляж** | 2 | 3 | 21 / 15 |
| **Снежный** | 2 | 3 | 15 / 15 |
| **Зал из 3х партий** | 2 | 3 | 25 / 15 |

## ⚠️ Известные проблемы

### Проблема: Вариант "Зал из 3х партий" не сохраняется

**Описание проблемы:**
При выборе варианта волейбола "Зал из 3х партий" (indoor-3sets) в настройках матча и сохранении, вариант не применяется корректно — после сохранения возвращается значение по умолчанию ("Зал" / indoor).

**Проявления:**
- При выборе "Зал из 3х партий" и нажатии "Сохранить" настройки сохраняются, но variant остаётся "indoor"
- При повторном открытии настроек матча выбрано "Зал" вместо "Зал из 3х партий"
- Количество партий остаётся 5 вместо 3

**Что было сделано для исправления:**

1. **Добавлен новый вариант в конфигурацию** (`volleyballRulesConfig.ts`):
   - Создан вариант `INDOOR_3SETS` с параметрами: 2 победы, макс 3 партии, 25/15 очков
   - Добавлен в `VARIANTS` и `RULES_CONFIGS`

2. **Обновлен тип Match** (`types/Match.ts`):
   - Добавлен `'indoor-3sets'` в тип variant интерфейса Match

3. **Добавлен в UI настроек матча** (`MatchSettingsPage.tsx`):
   - Добавлена опция в выпадающий список
   - Добавлена валидация variant при сохранении

4. **Исправлена синхронизация состояния**:
   - Добавлена зависимость `match?.variant` в useEffect для обновления formData
   - Добавлена логика обновления formData при изменении match.variant
   - Исправлено обновление match из location.state

5. **Добавлено логирование**:
   - Логи в handleSave для отслеживания значения variant
   - Логи в setCurrentMatch для проверки сохранения

**Текущий статус:**
Проблема требует дополнительного исследования. Возможно, проблема в том, что match не обновляется правильно после setCurrentMatch или в том, что formData не успевает обновиться перед сохранением.

**Временное решение:**
Используйте вариант "Пляж" и вручную изменяйте очки до 25 в настройках, либо используйте стандартный вариант "Зал" (до 5 партий).

## 🎯 Быстрый старт

1. **Запустите приложение** — откроется главная страница
2. **Создайте новый матч** — нажмите кнопку "Создать новый матч" или используйте `Ctrl+N`
3. **Настройте матч** — перейдите в "Настройки матча" и укажите:
   - Названия команд
   - Цвета формы команд
   - Информацию о турнире
   - Логотипы команд (опционально)
   - **Вариант волейбола** (Зал / Пляж / Снежный / Зал из 3х партий)
4. **Настройте vMix** (если используется):
   - Откройте «Вид» → «Настройки vMix»
   - Укажите IP адрес и порт, нажмите «Подключить»
   - Добавьте инпуты типа GT из vMix и сопоставьте поля с данными матча
5. **Начните ведение счета** — используйте кнопки на странице "Управление матчем"

📖 **Подробная инструкция по использованию:** см. [USER_GUIDE.md](USER_GUIDE.md)

## 📦 Основные функции

### Управление счетом
- Ведение счета текущей партии
- Автоматический учет перехода подачи при начислении очка
- Ручная коррекция подачи
- Отображение сетбола и матчбола
- Завершение партий с сохранением результата

### Интеграция с vMix
- Сохранённое состояние подключения (Подключено/Отключено)
- Динамические инпуты: список GT-инпутов из vMix
- Автоматическое обновление данных в vMix при изменении матча
- Поддержка 8 оверлеев, управление видимостью плашек
- Оптимизированная отправка данных (только изменённые поля)

### Управление матчами
- Создание нового матча
- Открытие существующего матча из файла
- Автосохранение матча (включено по умолчанию)
- Сохранение матча в файл (JSON формат)
- Быстрая смена команд местами

### Составы команд
- Загрузка и редактирование составов команд
- Указание позиций игроков (OH, MB, OPP, S, L)
- Выбор стартового состава
- Загрузка логотипов команд
- Указание тренеров команд

### Мобильный доступ
- Встроенный HTTP сервер для локальной сети (порт 3000)
- Доступ к контрольной панели с мобильных устройств
- QR-код для быстрого подключения
- Уникальные ссылки для безопасности

### Расширенная статистика
- Ведение статистики по категориям:
  - Атака
  - Блок
  - Подачи
  - Ошибки соперника
- Отображение статистики на плашках vMix

### ТВ-табло
- Отдельная страница для вывода на второй экран
- Крупное отображение счета (15rem)
- Синхронизация в реальном времени через мобильный сервер
- Ссылка: `http://[IP]:3000/tv`

### Тайм-ауты
- 2 тайм-аута на партию для каждой команды
- Таймер обратного отсчета (30 секунд)
- Автоматическое завершение по таймеру
- Визуальный прогресс-бар
